Mathematical Foundation Overview

The BrowserCoordinateKit library is based on a rigorous mathematical framework for calculating positions in browser windows, taking into account display and browser properties:

Coordinate Systems

  1. Screen Coordinates (S): Physical pixel positions on the screen, with origin at the top-left
  2. Normalized Coordinates (N): Screen-size independent positions in the range [0,1]
  3. Browser Coordinates (B): Physical pixel positions relative to the browser window top-left
  4. Logical Coordinates (L): Browser-internal coordinates after DPI scaling

Key Transformations

  • Screen ↔ Normalized: \(T_{S \to N}(p_s) = (x_s/s_w, y_s/s_h)\)
  • Screen ↔ Browser: \(T_{S \to B}(p_s) = p_s - b = (x_s - b_x, y_s - b_y)\)
  • Browser ↔ Logical: \(T_{B \to L}(p_b) = p_b/\sigma = (x_b/\sigma, y_b/\sigma)\)
